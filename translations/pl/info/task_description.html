Witamy w szklarni! Państwa zadaniem jest stworzenie "inteligentnego" asystenta, który będzie w stanie udzielić idealnych porad dotyczących podlewania różnych rodzajów roślin. Napiszą Państwo funkcję, która będzie mózgiem tego asystenta.

Zasady botaniki (i logiki funkcji):

  1. Reguła bezpieczeństwa - Jeśli gleba jest już wilgotna ({% if is_js %}isSoilDamp{% else %}is_soil_damp{% endif %} jest {% if is_js %}t{% else %}T{% endif %}rue), nigdy nie należy podlewać rośliny, niezależnie od innych warunków.

  2. Sukulenty ("Succulent" ) - Te pustynne rośliny są wrażliwe na nadmierne podlewanie. Nie należy ich podlewać, jeśli były podlewane w ciągu ostatnich 12 dni.

  3. Rośliny tropikalne ("Tropical" ) - Rośliny te uwielbiają wilgoć. Proszę je podlewać, jeśli od ostatniego podlewania minęły więcej niż 3 dni.

  4. Wszystkie inne rośliny ("Default" ) - W przypadku każdego innego rodzaju roślin należy postępować zgodnie z ogólną zasadą: podlewać je, jeśli od ostatniego podlewania minęło więcej niż 7 dni.

  5. Obsługa błędów - jeśli {% if is_js %}plantType{% else %}plant_type{% endif %} nie jest jednym z obsługiwanych typów, proszę zwrócić odpowiedni komunikat.

  6. Przypadki graniczne - jeśli jest to ostatni dzień okresu, zalecamy sprawdzenie następnego dnia.

W oparciu o zestaw reguł funkcja musi zwrócić ciąg znaków z prawidłową poradą: "Do not water", "Water the plant", "Unknown plant type" lub "Check the soil again tomorrow".
